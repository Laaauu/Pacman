# pacman
 Une simple application pour jouer à Pacman sur téléphone pour le GameJam 2026 du CESI.

 Installation du jeu :

    - Depuis GitHub sur votre téléphone, cliquez sur Releases Latest.

    - Téléchargez le fichier pacman.apk (ou appuyez dessus).

    - Sur votre téléphone, ouvrez le dossier Téléchargements et appuyez sur pacman.apk pour installer le jeu.
